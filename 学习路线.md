# C++ 后端开发学习路线

### 计算机基础

- **算法/数据结构**
  - 高频考点：链表、树、堆、排序查找
  - 学习工具：LeetCode 是必经之路

- **网络**
  - 重点协议：TCP/IP（三次握手/四次挥手）
  - Web协议：HTTP/HTTPS（版本区别、安全机制）

- **操作系统（Linux）**
  - 基础命令掌握
  - 日志排查技能
  - 简单的 Shell 脚本编写

- **设计模式**
  - 常考模式：单例、工厂、策略、观察者模式

### 语言深度（C++）
- **内存管理**
  - 内存模型（内存池/堆栈）
  - 虚函数表（多态实现）
  - RAII（资源获取即初始化）
  - 智能指针原理

- **标准库**
  - STL 源码实现

- **多线程编程**
  - 理解进程/线程/协程关系
  - 并发三大问题：
    - 原子性
    - 可见性
    - 有序性
  - 锁机制：
    - 互斥锁
    - 自旋锁
    - 读写锁

### 开发工具与框架

- **必备工具**
  - Git（版本控制）
  - Postman（接口测试）
  - IDE（开发环境）

- **框架理解**
  - C++ 后端框架：
    - Web Server 框架
    - RPC 框架（如 gRPC, brpc）
  - 掌握 CRUD（增删改查）的基本开发逻辑

### 数据存储

- **MySQL**
  - 索引原理（B+ 树）
  - 事务 ACID 特性
  - 隔离级别
  - MVCC（多版本并发控制）
  - 锁机制

- **Redis（高性能关键）**
  - 5 种基本数据结构
  - 持久化机制：
    - RDB
    - AOF
  - 缓存问题解决方案：
    - 缓存穿透
    - 缓存击穿
    - 缓存雪崩
  - 分布式锁实现

### 加分项

- **消息队列**
  - Kafka/RabbitMQ
  - 应用场景：
    - 系统解耦
    - 削峰填谷

- **容器化**
  - Docker
  - Kubernetes（云原生基础）

- **进阶技术**
  - 分布式锁
  - 限流算法
  - 布隆过滤器
  - 加密算法
